## 4. 注意事项

- 论文使用 RoBERTa-large，本实验使用 SciBERT（与当前实验一致），结果不直接可比
- 论文的 Micro-F1/Macro-F1 报告口径可能包含 overall 指标，需确认
- 当前实验使用 143 child labels (Data.xlsx)，HYDRA 使用 134 child labels (Y.txt)，数据划分可能不同
- 如需严格公平对比，建议使用相同数据划分和相同标签口径重新跑当前方法
